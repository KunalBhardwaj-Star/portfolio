### Dependencies install

1. npm create vite@latest portfolio -- --template react-ts
    Create React + TypeScript folder

2. npm install tailwindcss @tailwindcss/vite
    tailwind CSS

3. npm install drizzle-orm postgres
    drizzle-orm → ORM
    postgres → PostgreSQL driver used by Drizzle

4. npm install -D drizzle-kit dotenv
    drizzle-kit → migrations and schema generation
    dotenv → environment variables

5. npm install @supabase/supabase-js
   Supabase Client
